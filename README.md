# Rental Property Management System

A Python-based financial tracking tool designed to streamline the management of multiple rental properties. This system acts as a centralised digital ledger, automating the reconciliation of rent collection, maintenance costs, and mortgage obligations to provide clear insight into property profitability.

Managing the finances of rental properties manually can lead to errors and a lack of clarity. This project solves that problem by:
* **Centralising Data:** Replacing manual notes with a structured digital log.
* **Automating Math:** Automatically calculating net cash flow (Revenue - Expenses).
* **Performance Tracking:** Comparing actual revenue against mortgage obligations to determine immediate profitability.

* **Transaction Logging:** Efficient input of granular financial details (Rent received, Repair costs, Maintenance fees).

* **Data Integrity:** Uses input validation to ensure financial records are accurate and consistent.
